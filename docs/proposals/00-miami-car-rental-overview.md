## Por que Neero

### Codigo en Produccion, No Promesas

| Proyecto     | Componente reutilizado                         |
| ------------ | ---------------------------------------------- |
| migue.ai     | WhatsApp API v23, Google Calendar, cron reminders |
| cobru.ai     | WhatsApp Flows v6 (formulario nativo en app)   |
| drad         | Dashboard admin, data tables, filtros, exports |
| tip2025      | RBAC, Supabase Auth, admin CRUD                |

Estos proyectos estan en produccion. Las integraciones criticas ya existen y estan probadas.

### Diferenciadores Clave vs TENET

| Diferenciador                  | Descripcion                                                              |
| ------------------------------ | ------------------------------------------------------------------------ |
| WhatsApp Flows nativo          | Reserva directamente en WhatsApp sin abrir navegador (cobru.ai lo tiene) |
| Sin Twilio                     | WhatsApp Business API directa - mensajes de servicio GRATIS desde Nov 2024 |
| Sin Payload CMS                | Admin nativo Next.js + Supabase, sin licencias ni overhead               |
| AI-Ready                       | Vercel AI SDK 5.0 + OpenAI listo para Fase 2 conversacional              |
| 1 semana mas rapido            | Reutilizacion directa acelera entrega                                    |
| Menor costo mensual            | ~$25/mes menos (eliminando Twilio)                                       |
