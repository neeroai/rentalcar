# Demo script

## 1. Abrir homepage

- Mostrar el hero full-bleed, el toggle ES/EN y el buscador.
- Explicar que el producto elimina counters y categorías genéricas.

## 2. Ejecutar búsqueda

- Buscar desde MCO con fechas demo.
- Destacar filtros, sticky summary y hover en resultados.

## 3. Abrir ficha de vehículo

- Resaltar galería, reputación del host, políticas y delivery options.
- Mostrar que el rail de reserva mantiene el contexto del viaje.

## 4. Completar checkout

- Revisar summary, add-ons y UI de pago mock.
- Confirmar la reserva y navegar a la página de éxito.

## 5. Mostrar post-booking

- Abrir `booking/confirmed`.
- Ir a `account/messages` para enseñar coordinación host-huésped.
- Ir a `account/trips` para cerrar el loop.

## 6. Mostrar lado host

- Abrir `/host`.
- Usar la earnings calculator.
- Entrar a `/host/list-your-car` y avanzar en el wizard.

## 7. Cierre

- Enfatizar que el prototipo es navegable, bilingüe, demo-ready y orientado a pitch.
- Explicar qué falta para pasar a MVP de producción: backend, auth, pagos, disponibilidad real y contenido final del cliente.
