## Pricing Model Recomendado para Independiente Nuevo

### Posicionamiento Estrategico

| Estrategia      | Posicion vs Enterprise                    | Posicion vs Turo                       | Racional                                                   |
| --------------- | ----------------------------------------- | -------------------------------------- | ---------------------------------------------------------- |
| Precio base     | -10% a -15% bajo enterprise               | Paridad o +5% sobre Turo               | Atraer por precio vs enterprise; servicio superior vs Turo |
| Propuesta valor | Servicio personalizado + sin fees ocultos | Vehiculo garantizado + seguro incluido | Diferenciacion por transparencia y confiabilidad           |

**Racional:** Independientes pequenos historicamente compiten 10–15% bajo grandes cadenas para compensar menor reconocimiento de marca y menos ubicaciones. (Fuente: NerdWallet Hertz vs Enterprise analysis; AutoSlash independent rental analysis)

## Estrategia de Descuentos

| Tipo descuento                 | % Recomendado            | Industria benchmark                | Racional                                                                    |
| ------------------------------ | ------------------------ | ---------------------------------- | --------------------------------------------------------------------------- |
| Semanal (5–7 dias)             | 15–20% off diaria        | Hasta 25–35% (Avis, Budget)        | Menor que enterprise para proteger margen; flota pequena no puede subsidiar |
| Quincenal (8–14 dias)          | 20–25% off diaria        | N/D estandar                       | Incentivo intermedio para turistas largo                                    |
| Mensual (28+ dias)             | 25–35% off diaria        | 10–50%+ (Enterprise Month-or-More) | Ocupacion garantizada vs margen; ideal snowbirds                            |
| Early booking (14+ dias antes) | 5–10% adicional          | Hasta 40% ahorro (industria)       | Incentiva reserva anticipada, mejor planificacion                           |
| Repeat customer                | 10% off siguiente rental | Programas loyalty enterprise       | Construccion de base de clientes leales                                     |
| Direct booking (web propia)    | 5% off vs OTA            | Estandar industria                 | Evita comision OTA 15–25%                                                   |

**Fuentes:** Avis weekly deal (hasta 25%), Budget weekly deal (hasta 35%), Enterprise Month-or-More (hasta 58% off), AutoSlash long-term analysis 2025.
