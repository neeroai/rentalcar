| Plataforma     | Tipo            | Comision Car Rental       | Notas                                  |

| Expedia        | OTA             | 10-30% (tipico 15-20%)    | Pay-per-book, independientes pagan mas |
| Booking.com    | OTA             | 10-25%                    | Similar a Expedia                      |
| Kayak          | Metasearch      | No cobra comision directa | Redirige a OTA o sitio del proveedor   |
| Rentalcars.com | Agregador       | 10-17%                    | Especializado en car rental            |
| Google Travel  | Metasearch      | CPC model                 | No comision, pago por click            |
| Turo           | P2P marketplace | 25-40% (host fee)         | Modelo peer-to-peer                    |

(Fuentes: stayfi.com/vrm-insider/ota-fees; altexsoft.com/blog/ota-rates-commission-engine; cloudbeds.com, consulta 2026-03-10)

### Analisis Costo-Beneficio OTAs

| Factor            | Ventaja                         | Desventaja                                         |
| ----------------- | ------------------------------- | -------------------------------------------------- |
| Volumen           | Acceso a millones de viajeros   | Comision 10-17% por reserva                        |
| Visibilidad       | Listing en plataformas globales | Competencia directa con grandes cadenas            |
| Sin costo upfront | Solo paga cuando hay reserva    | Margen reducido                                    |
| Reviews/ratings   | Credibilidad para negocio nuevo | Dependencia de plataforma                          |
| Control de precio | Puede ajustar tarifas           | OTA puede mostrar precios mas bajos de competencia |

### Recomendacion OTAs

| Accion                                              | Prioridad | Impacto                             |
| --------------------------------------------------- | --------- | ----------------------------------- |
| Registrarse en Expedia + Kayak + Rentalcars.com     | MEDIA     | Volumen inicial, visibilidad        |
| Mantener precio directo 5-10% mas bajo que OTA      | ALTA      | Incentivar reserva directa          |
| Usar OTAs para adquisicion, WhatsApp para retencion | ALTA      | Reducir dependencia OTA             |
| Google Travel listing                               | MEDIA     | Alternativa sin comision (solo CPC) |
