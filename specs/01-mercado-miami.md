## 6. Impacto de Plataformas Peer-to-Peer (Turo/GetAround)

### 6.1 Turo -- Posicion de Mercado

| Metrica           | Valor                                   | Fuente   |
| ----------------- | --------------------------------------- | -------- |
| Revenue Turo 2024 | ~$958M                                  | Sacra    |
| Crecimiento YoY   | ~9% (desaceleracion desde 213% en 2021) | Sacra    |
| Vehiculos activos | ~350,000 en 16,000+ ciudades            | Turo S-1 |
| Usuarios          | 14+ millones                            | Turo     |
| Rentals mensuales | ~1.5 millones                           | Turo     |

### 6.2 Contexto Competitivo P2P vs Tradicional

| Empresa             | Revenue 2024                | Nota                     |
| ------------------- | --------------------------- | ------------------------ |
| Enterprise Holdings | #1 global                   | Mayor flota USA          |
| Avis Budget         | $11.8B (perdida neta $1.8B) | Sacra                    |
| Hertz               | $9.0B (perdida neta $2.9B)  | Sacra                    |
| Turo                | $958M                       | Unico P2P major restante |

- GetAround cerro operaciones USA en 2024 (Sacra)
- Uber Carshare descontinuado; Uber firmo partnership con Turo para integrar en app Uber (Sacra)
- Turo abrio su plataforma a empresas de rental independientes como "Commercial Hosts", permitiendo retener 90% de earnings con seguro comercial propio (Auto Rental News)

### 6.3 Implicaciones para Independiente en Miami

| Factor                                | Impacto  | Oportunidad                                          |
| ------------------------------------- | -------- | ---------------------------------------------------- |
| Turo como canal de distribucion       | POSITIVO | Listar vehiculos como Commercial Host (90% earnings) |
| Acceso a 150M usuarios via Uber-Turo  | POSITIVO | Demanda masiva sin costo de adquisicion              |
| Competencia precio con fleets grandes | NEGATIVO | Grandes con pricing agresivo post-pandemia           |
| Regulacion P2P incierta               | RIESGO   | Cambios regulatorios posibles en Miami-Dade          |

### Factores de Crecimiento

| Factor                  | Tendencia                            | Impacto Miami                 |
| ----------------------- | ------------------------------------ | ----------------------------- |
| Turismo post-pandemia   | Consolidacion (crecimiento moderado) | Positivo                      |
| Trabajo remoto/bleisure | Estancias mas largas                 | Positivo (mas dias de rental) |
| Electrificacion flotas  | Adopcion gradual (4% penetracion)    | Neutro a corto plazo          |
| Integracion Turo-Uber   | Expansion demanda P2P                | Positivo para independientes  |
| Regulacion              | Surcharges, seguros, licensing       | Costo operativo               |
