### Regla Clave: Seguro Comercial Propio

Hosts que proveen su propio seguro comercial (opcion para empresas de car rental):

- Retienen **80% del precio de viaje** (fijo, no variable)
- **No reciben** proteccion de Turo (ni liability ni dano fisico)
- Deben procesar claims via su propia aseguradora
- Modalidad ideal para empresas ya aseguradas con poliza comercial

_Fuente: selectriskgroup.com/commercial-auto-insurance-for-turo-hosts — 2024_

## 3. Requisitos Vehiculos para Listar en Turo

### Criterios de Elegibilidad

| Criterio                 | Requisito                                                 |
| ------------------------ | --------------------------------------------------------- |
| Edad maxima              | 12 anos de antiguedad (modelo 2014 en adelante en 2026)   |
| Millaje maximo al listar | Menos de 130,000 millas                                   |
| Titulo                   | Clean title obligatorio — sin salvage, rebuilt, o branded |
| Valor de mercado         | Maximo $200,000 (limite cobertura Turo)                   |
| Recalls de seguridad     | Ninguno pendiente — prohibido listar                      |
| Capacidad maxima         | 10 personas incluyendo conductor                          |
| Registro                 | A nombre del host o con autorizacion documentada          |
| Placas                   | Permanentes o temporales autorizadas (no dealer plates)   |
| Seguro personal          | Requerido por ley, aunque Turo cubre durante viajes       |
| Tracker GPS              | Obligatorio si valor >= $125,000 (OEM tracker)            |

### Restricciones de Modelos

| Tipo de Vehiculo                          | Elegibilidad Turo                               |
| ----------------------------------------- | ----------------------------------------------- |
| Autos de pasajero (sedan, SUV, hatchback) | Elegible                                        |
| Minivans                                  | Elegible                                        |
| Trucks pickup                             | Elegible                                        |
| Specialty / Classic (>12 anos pero <25)   | Elegible con revision manual, valor max $85,000 |
| Vehiculos comerciales (box truck, cargo)  | No elegible                                     |
| Vehiculos con >10 asientos                | No elegible                                     |
| Autos con salvage/rebuilt title           | No elegible                                     |

### Requisitos Especificos Florida

Florida no tiene restricciones adicionales significativas vs resto de USA. Responsabilidades del host:

- Cumplir registro y seguro obligatorio del estado de Florida
- Florida no requiere inspeccion de emisiones (a diferencia de NY, CA)
- Verificar con help.turo.com/en_us/vehicles-we-accept-rylmrNl45 para actualizaciones

_Fuente: help.turo.com/en_us/vehicles-we-accept-rylmrNl45 — 2025_

### Clausula Grandfather

Si el vehiculo fue elegible al momento de listarlo, puede continuar en la plataforma incluso si supera los 130,000 millas, siempre que se mantenga en excelentes condiciones y cumpla todos los estandares de seguridad y mantenimiento.

Turo partnership con **Tint Insurance**: cobertura para vehiculos entre reservas cuando se usan para negocio.

- Ideal para flotas comerciales que tambien usan vehiculos para operaciones propias
- Costo adicional — verificar rates en turo.com/host

- Hosts establecen precio base por dia

## 8. Integracion Operacional

### Modelo de Gestion Paralela: Turo + Directo

| Desafio                      | Solucion                                                                                     |
| ---------------------------- | -------------------------------------------------------------------------------------------- |
| Calendario doble reservas    | Bloquear dias en Turo cuando hay reserva directa y viceversa (app nativa)                    |
| Entrega/devolucion diferente | Definir protocolo unico: foto pre/post, checklist, llave entregada via lockbox o en persona  |
| Precios divergentes          | Politica: Turo = precio mercado (sin descuento excesivo), Directo = precio premium o paquete |
| Limpieza entre reservas      | Scheduler de cleaning service entre viajes — prioridad alta en temporada pico                |
| Danos y claims               | Documentar TODO: fotos timestamp pre-trip via app Turo + checklist fisico firmado            |

### Flujo de Operacion Recomendado

```
RESERVA TURO                     RESERVA DIRECTA
     |                                  |
Bloquear calendario               Bloquear Turo
propio para ese vehiculo          para esas fechas
     |                                  |
Foto pre-trip via app             Contrato propio

     |                                  |
Entrega: lockbox MCO              Entrega: oficina
o en ubicacion acordada           o MCO pickup
     |                                  |
```

## 10. Limitaciones y Riesgos

### Riesgos de Dano al Vehiculo

| Riesgo                           | Probabilidad | Costo Potencial                |
| -------------------------------- | ------------ | ------------------------------ |
| Dano exterior (rasguno, abollon) | Media-Alta   | $500-$2,000                    |
| Accidente mayor                  | Baja         | $5,000-$50,000+                |
| Total loss                       | Muy Baja     | ACV del vehiculo               |
| Dano interior (mancha, olor)     | Media        | $200-$800                      |
| Robo del vehiculo                | Muy Baja     | ACV del vehiculo               |
| Mal uso / dano mecanico          | Baja         | Variable, NO CUBIERTO por Turo |

Dato real: _Un host Turo reporto que un renter total-loss un Audi R8 y quedo con $16,000 de diferencia entre ACV pagado y costo real._

### Politica de Cancelaciones

| Tipo                     | Detalle                                                     |
| ------------------------ | ----------------------------------------------------------- |
| Free cancellation host   | No aplica — Turo penaliza cancelaciones de host             |
| Penalizacion host cancel | Bloqueo de fechas, posible reduccion de ranking en busqueda |
| Guest cancellation       | Segun politica elegida por host (flexible/moderate/strict)  |
| Impacto All-Star         | Cancelaciones de host afectan directamente el status        |

### Restricciones de Uso Comercial en Terminos Turo

| Restriccion   | Detalle                                                                |
| ------------- | ---------------------------------------------------------------------- |
| Rideshare     | Guests NO pueden usar vehiculo Turo para Uber/Lyft — terminos prohiben |
| Subletting    | Host no puede re-listar vehiculos de terceros sin permiso Turo         |
| Ubicacion     | Vehiculo debe operar en la ciudad/zona donde fue listado               |
| Customizacion | Restricciones en branding del vehiculo segun plan                      |
| Disputas      | Turo usa **binding arbitration** — no litigation directa en corte      |

### Impacto en Seguro Comercial Propio

| Situacion                    | Riesgo                                                                 |
| ---------------------------- | ---------------------------------------------------------------------- |
| Host usa seguro personal     | Aseguradora puede **NEGAR** claim por uso comercial                    |
| Costo promedio sin cobertura | $4,200 por incidente (out-of-pocket)                                   |
| Poliza comercial propia      | Recomendada para flota 3+ vehiculos — costo $400-$800/ano por vehiculo |
| Fleet policy 3+ vehiculos    | Reduce costo administrativo + cobertura uniforme toda la flota         |

### Limitaciones Operacionales Adicionales

| Limitacion              | Impacto                                                                   |
| ----------------------- | ------------------------------------------------------------------------- |
| Depreciacion acelerada  | Vehiculos Turo se deprecian 2-3x mas rapido que en uso normal             |

| Fondo de reemplazo      | Expertos recomiendan reservar $2,000/vehiculo/ano para reemplazo          |

| IPO cancelado 2025      | Senales de desaceleracion estrategica — monitorear estabilidad plataforma |
| Arbitraje obligatorio   | Disputas con Turo requieren arbitraje vinculante — NO litigacion          |
